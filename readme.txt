工具简介:
    解决的问题：两地开发时，对文档，代码，以及编译生成的二进制文件的版本管理。
    特点：
    1.该工具用脚本编写（ruby），便于使用人员的需求随时修改随时执行。
    2.文件的比较基于CRC32文件校验技术，比较的结果不受文件修改时间的影响。

注意事项：
*****************************************************
*使用前应该保证您的电脑装有RUBY的执行环境。         *
*如果没有，请前往https://www.ruby-lang.org/下载。   * 
*Windows用户建议下载rubyinstaller的安装包。         *
*安装过程，简单方便，不再赘述。                     *
*****************************************************

ready_update_file.sh :用于对指定项目的添加管理。添加管理后项目可以在history中找到变更的履历
copy_update_file_to_home:用于从history中汇集总变更文件到home目录
copy_to_remote_server:用于将home目录下的变更文件，拷贝至远端的服务器，并通知远端服务器进行部署
