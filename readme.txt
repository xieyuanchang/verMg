工具简介:
    解决的问题：两地开发时，对文档，代码，以及编译生成的二进制文件的版本管理。
    特点：
    1.该工具用脚本编写（ruby），便于使用人员的需求随时修改随时执行。
    2.文件的比较基于CRC32文件校验技术，比较的结果不受文件修改时间的影响。

注意事项：
*****************************************************
*使用前应该保证您的电脑装有RUBY的执行环境。         *
*如果没有，请前往https://www.ruby-lang.org/下载。   * 
*Windows用户建议下载rubyinstaller的安装包。         *
*安装过程，简单方便，不再赘述。                     *
*****************************************************

使用方法：
1.修改VerMg目录下的附带的init.bat文件
    修改方式如下：
    ruby VerMg.rb 参数 #将您需要管理的文件夹作为参数（复数可）传给VerMg.rb处理即可。

2.执行修改后的start.bat文件
    首次执行时，目标文件夹会被注册为管理对象文件夹,并将在verInfo目录下留下记录。
    下次执行时，如果已经注册为管理对象文件夹内的文件曾被修改，将生成history目录，
    并在文件夹内创建一个对应的差分文件夹。
